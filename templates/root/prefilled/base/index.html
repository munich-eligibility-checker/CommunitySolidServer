M-Pod Server

Willkommen bei M-Pod

Dieser Server implementiert das Solid-Protokoll, damit Sie Ihren eigenen Solid Pod und Ihre eigene Identität erstellen können.

Erste Schritte als Benutzer

Registrieren Sie sich für ein Konto, um mit Ihrem eigenen Pod und Ihrer WebID zu starten.

Die Registrierung ist auf diesem Server deaktiviert. Stellen Sie sicher, dass Sie die Berechtigungen des Root-Containers aktualisieren, um zu verhindern, dass andere Personen Ihren Server modifizieren.

Um mehr darüber zu erfahren, wie dieser Server verwendet werden kann, schauen Sie sich das Erste-Schritte-Tutorial und die Server-Dokumentation an.

Erste Schritte als Entwickler

Sie können jede der Konfigurationen im config-Ordner des Servers verwenden, um eine Instanz dieses Servers mit verschiedenen Funktionen einzurichten. Neben den bereitgestellten Konfigurationen können Sie auch Ihre eigene benutzerdefinierte Konfiguration mit dem Konfigurations-Generator erstellen.

Sie können einfach jeden Ordner auf Ihrer Festplatte als Root-Pod freigeben.
Verwenden Sie den --help-Schalter, um mehr zu erfahren.

Viel Erfolg mit Solid

Erfahren Sie mehr über Solid unter solidproject.org.

Sie sind herzlich eingeladen, Ihre Erfahrungen zu teilen und Fehler zu melden, die Sie finden.
